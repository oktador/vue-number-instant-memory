# vue-number-order
Numbers from 1 to n (parameter) will be shown on the screen for m (parameter) ms.
Then the numbers wont be shown, and you have to click the numbers sequentially.
Test your memory.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
